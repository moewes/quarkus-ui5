== Installation

If you want to use this extension, you need to add the `io.quarkiverse.ui5:quarkus-ui5` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.ui5</groupId>
    <artifactId>quarkus-ui5</artifactId>
</dependency>
----
